# VerticalViewPager

### 垂直滚动的ViewPager，基于support-v4:ViewPager:26.1.0修改

## 使用
```
allprojects {
    repositories {
        google()
        jcenter()
        maven { url 'https://dl.bintray.com/1109993488/maven' }
    }
}

dependencies {
    implementation 'com.blingbling.widget:vertical-view-pager:1.0.0'
}
```
